# DataX

DataX 是阿里云 DataWorks数据集成 的开源版本，在阿里巴巴集团内被广泛使用的离线数据同步工具/平台。

项目传送门: https://github.com/alibaba/DataX

# DataX Web

DataX Web 是基于 DataX 开发的一个 Web 访问套件。



#### 使用方法：

```
git clone https://github.com/gindowcloud/datax

cd datax

docker-compose up
```



#### 访问地址

http://127.0.0.1:8000

用户: admin

密码: admin
